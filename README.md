The Excel files M5_Level12.xlsx, M5_Level11.xlsx and M5_Level10.xlsx provide the time series of dataset 1 as used in Sbrana & Babai (2026, EJOR).

The dataset 1 refers to the M5 competition data available on Kaggle website (see: https://www.kaggle.com/c/m5-forecasting-accuracy/overview) 

For storage reasons we saved each time series in one cell using a string for all observations.

For each series the last 28 observations are the out-of-sample data, and the remaining initial observations are the in-sample data. 

Therefore, in-sample and out-of-sample data are concatenated in a single string.

The Excel file dataset2.xlsx provides the dataset 2 as used in Sbrana & Babai (2026, EJOR). 

The file contains four different spreadsheets: steps= 14, steps= 28, steps= 56, Raw data

While the Raw data refer to the original data, the other spreadsheets contain the series used to obtain the results in Sbrana & Babai (2026, EJOR)

Again, for storage reasons we saved each time series in one cell using a string for all observations.

For each series the last 14, 28, 56 observations are, respectively, the out-of-sample data, and the remaining initial observations are the in-sample data. 

Therefore, in-sample and out-of-sample data are concatenated in a single string.
